Block Elements

By default, a block-level element occupies the entire space of its parent element (container), thereby creating a "block." A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).

Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary.

Few Inline elements are:

Buttons, inputs, img, i(italic), span, b(bold), iframe, lables, progress, strong and em(emphasis), anchors
